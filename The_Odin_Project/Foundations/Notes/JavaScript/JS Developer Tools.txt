Lesson Overview:

- Change screen size of website with developer tools
- View and change the DOM
- Debug JavaScript
- Use breakpoints
- View and edit HTML in the Elements tab
- View the Resources Panel to check the scripts running on the website
- Add CSS Pseudostate to a Class
- View CSS properties in Alphabetical Order 
- View and edit the Box Model of any Element in Chrome DevTools
- View a page in print mode 
- Enable and Disable CSS classes
- Simulate media queries in Device Mode 
-------------------------------------------------------------------------------------------------

Opening DevTools:

There are three ways to do this.
- From chrome menu > More Tools > Developer Tools
- Right-click anywhere on a webpage and select Inspect
- Keyboard shortcut F12 or Ctrl + Shift + C (Mac: Opt + Cmd + C)
